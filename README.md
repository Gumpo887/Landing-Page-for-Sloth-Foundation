 Description (EN)

Modern, responsive, and accessible landing page for the Sloth Rescue and Conservation Foundation, built with:

Languages: TypeScript / JavaScript (.ts, .tsx in src/)

UI framework: React 18 (react, react-dom)

Bundler / dev server: Vite with @vitejs/plugin-react-swc (SWC-based React transform)

Styling: Tailwind CSS v4.1.3 (utilities like bg-emerald-600, rounded-2xl, backdrop blur, and modern CSS features, defined in index.css)

Components / primitives: Radix UI (@radix-ui/react-*), Lucide icons (lucide-react)

UI/UX libs: cmdk (command menu), embla-carousel-react (carousel), recharts (charts), sonner (notifications/toasts), input-otp

Forms: react-hook-form

Theming: next-themes for theme switching

Utilities: clsx, class-variance-authority, tailwind-merge, react-resizable-panels, vaul

Build / tooling: Full TypeScript support, vite.config.ts with aliases and build.target: 'esnext', SWC for fast transforms

The project is a pure front-end static landing page, focused on clean UX, accessibility, and a modern React/Tailwind developer experience.
